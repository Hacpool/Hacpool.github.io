How To Mine

Simple setup anyone can follow.

If your Ubuntu machine already has an NVIDIA driver, you only need a few packages, your Hacash wallet address and the miner files from this page.

1. Install required dependencies

Run the dependency script first. Then make sure your NVIDIA card is visible.

sudo bash ./install_deps_ubuntu.sh
nvidia-smi

2. Give execution permission

This only needs to be done once after downloading the files.

chmod +x ./poolworker ./start_miner.sh ./start_miner_single.sh

3. Start mining with all NVIDIA GPUs

Replace HACASH_WALLET with your own wallet address.

./start_miner.sh HACASH_WALLET rig01

# Single GPU example
./start_miner_single.sh HACASH_WALLET rig01 0

Required packages

The dependency script installs these runtime packages on Ubuntu:

  • ca-certificates
  • curl
  • libssl3
  • ocl-icd-libopencl1
